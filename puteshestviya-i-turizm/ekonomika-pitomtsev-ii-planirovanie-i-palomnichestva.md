# Экономика питомцев, ИИ-планирование и паломничества поп-культуры. *AMADEUS*

*Источник: AMADEUS - Travel Trends 2026.pdf*

Global. Шесть трендов, меняющих представление о путешествиях в 2026: экономика питомцев (животные путешествуют с достоинством), сочетание ИИ и человеческого опыта, узкофюзеляжные дальнемагистральные самолёты, паломничества поп-культуры, гостиничные номера на заказ, путешествия в технологическое будущее. 56% людей имеют домашнего питомца, почти половина из них (47%) — завели впервые. Индустрия домашних животных вырастет на 45% к 2030, достигнув 500 млрд долларов. 27% владельцев домашних животных, которые брали их в отпуск в 2025, сделали это впервые. Использование генеративного ИИ для планирования путешествий выросло на 64% в годовом сравнении (с 11% до 18%), но 25% путешественников получали устаревшую информацию, лишь 46% доверяют ИИ-системам.

#### Тренды
1. **Экономика питомцев** / The Pawprint Economy — 56% людей имеют домашнего питомца, индустрия вырастет до 500 млрд долларов к 2030; 27% владельцев брали питомца в отпуск впервые в 2025; барьеры — стресс для животного (42%), сомнения в приспособленности мест (21%), ограничения по направлениям (20%); новые правила: Китай разрешил питомцев на железных дорогах, Италия — средних собак в кабине самолёта, Великобритания вернула паспорта для питомцев; частные чартеры BarkAir и SkyePets запускают трансокеанские рейсы с питомцами в кабине; отели вроде AKA создают клубы для собак
2. **Сочетание ИИ и человеческого опыта** / Travel Mixology — использование генеративного ИИ для планирования выросло на 64% в годовом сравнении (с 11% до 18%), но 25% получали устаревшую информацию, лишь 46% доверяют ИИ; путешественники комбинируют языковые модели для общих идей, форумы и видео для живого опыта, ИИ-ассистенты брендов для персонализации; самые влиятельные источники — сарафанное радио (36%) и пользовательское видео (26%); бренды должны оптимизировать контент для ИИ-поиска и одновременно создавать подлинный человеческий контент
3. **Прямые перелёты** / Point-to-Point Precision — новые флоты дальнемагистральных узкофюзеляжных самолётов сжимают мир; Airbus A321XLR с дополнительным топливным баком увеличивает дальность на 700 морских миль; более 500 заказов, первые поставки во второй половине 2025; примеры новых маршрутов: Wizz Air Лондон-Джидда, IndiGo Индия-Афины впервые, Air Canada Монреаль-Мальорка впервые, Aer Lingus рост на 171% в Индианаполис и на 102% в Нэшвилл; топливная эффективность выше на 25-30% на место; Qantas свяжет Сидней-Лондон/Нью-Йорк без остановок, сократив время на 4 часа; на борту зоны для движения, здоровые закуски, программы упражнений, освещение для борьбы с джетлагом; 69% путешественников использовали бы биометрию вместо паспорта
4. **Паломничества поп-культуры** / Pop Culting — культурная интеллектуальная собственность становится драйвером туризма; развлекательные франшизы вдохновляют фанатские паломничества к местам из фильмов, сериалов, музыки; путешественники хотят физически пережить миры с экранов; бренды создают специальные пакеты на основе поп-культурных трендов
5. **Конструктор гостиничных номеров** / Pick 'n' Stays — отели дают путешественникам свободу выбирать каждую деталь номера: тренажёры Pilates, затемняющие шторы, мониторы для работы, завтрак поблизости; 63% готовы платить дополнительно за конкретные удобства; 17% бизнес-путешественников заплатили бы до 20% больше за бизнес-услуги, 12% поколения Z — до 25% за игровые консоли; Hotelverse позволяет осмотреть цифровой двойник отеля в 3D перед бронированием; разделение элементов проживания открывает новые возможности: наборы для здоровья, дизайнерские ароматы, аренда номеров для послеобеденного сна
6. **Инновационный туризм** / Innovation Tourism — путешественники посещают места, где будущее уже наступило; роботы для багажа, оплата жестом ладони, голографические проекции в метро, блюда из 3D-принтера, доставка дронами — это реальность в Дубае, Сингапуре, Сеуле и Токио; путешественники хотят увидеть, как выглядит будущее уже сегодня
